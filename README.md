## Project Overview
This repository contains the deliverables for the CSE0001 Digital Literacy course. 
Each folder corresponds to a specific module designed to enhance digital 
presence, communication, and security.

## Module Summaries
Module 1: Digital Awareness & Visual Communication

Objective: To create a visual resource that defines Digital Literacy and its importance for students.

Deliverable: A one-page infographic designed using a professional creative tool.
+1

Key Topics: Defining digital literacy, safe internet practices, and essential digital tools.

Module 2: Professional Digital Presence

Objective: To establish a foundational professional identity across major academic and industry platforms.
+1

Deliverable: Setup of professional profiles on GitHub and LinkedIn, including a customized GitHub profile README.

Focus: Learning how to present academic and technical skills to the global community.
+1

Module 3: Coding & Collaborative Tools

Objective: To build familiarity with platforms used for technical practice and cloud-based teamwork.

Deliverable: Completion of a beginner-level coding challenge and the creation of a digital literacy assessment quiz via Google Forms.
+1

Focus: Practical application of logic and data collection tools.

Module 4: Professional Communication & Etiquette

Objective: To master the art of professional correspondence and responsible social media behavior.

Deliverable: Drafting formal emails for academic and professional requests, and a comprehensive social media "Do’s and Don’ts" checklist.
+1

Focus: Understanding tone, structure, and digital accountability.

Module 5: Cyber Security & Ethical Awareness

Objective: To recognize, analyze, and prevent common online threats and frauds.

Deliverable: A detailed case study on a specific cybercrime and a "Stay Safe Online" prevention guide.
+1

Focus: Promoting financial/UPI safety and awareness of national reporting channels.
